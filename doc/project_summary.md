# 肤理通 (DermaScan AI) —— 项目详细摘要

## 1. 项目概览
**肤理通 (DermaScan AI)** 是一款基于 **视觉大模型 (Vision Foundation Models)** 与 **检索增强生成 (Retrieval-Augmented Generation, RAG)** 技术的下一代皮肤健康智能分析平台。项目旨在通过移动端便捷的图像采集与云端先进的人工智能算法，为用户提供专业的皮肤病变筛查、健康风险预警及个性化护理建议。

## 2. 核心技术栈
系统采用前后端分离的现代化架构，确保了系统的高可用性与可扩展性：

- **前端 (Frontend)**: 
  - 框架：React 18 + TypeScript + Vite
  - 样式：TailwindCSS + Framer Motion (精美动效)
  - 特效：移动优先设计，支持 **PWA (Progressive Web App)**，支持桌面安装与离线缓存。
- **后端 (Backend)**: 
  - 框架：Python FastAPI (异步高性能)
  - 数据库：SQLAlchemy + SQLite (轻量化部署)，Alembic 版本控制。
  - 鉴权：JWT 状态守卫。
- **AI 引擎层**:
  - **视觉识别**: **DermFM-Zero**。基于视觉大模型的零样本 (Zero-shot) 学习能力，无需特定微调即可识别多种皮肤临床异常。
  - **知识检索**: **FAISS** + **SentenceTransformers**。构建向量索引库，实现毫秒级的医学知识片检索。
  - **内容生成**: **阿里云通义千问 Qwen-Max**。负责将视觉推理结果与医学知识进行语义融合，输出结构化报告。

## 3. 核心功能模块
1. **AI 皮肤识别**: 
   - 支持拍照预览与本地图片上传。
   - 实现秒级疾病分类，输出带有置信度的 Top-k 诊断结果。
2. **RAG 医疗报告引擎**: 
   - 传统大模型易产生“幻觉”，本项目通过检索专业医学百科，确保生成的建议具有医学依据。
   - 报告包含：症状解析、潜在成因、日常护理及就医建议。
3. **皮肤助手 (Medical Assistant)**: 
   - 交互式 AI 咨询，基于 RAG 技术回答用户的皮肤健康疑问。
4. **个性化档案与趋势**: 
   - 记录用户识别历史，结合用户个人画像（年龄、历史病史）提供精细化的健康管理方案。
5. **皮肤百科检索**: 
   - 内置上千条垂直领域的皮肤疾病词条，支持离线/在线模糊搜索。

## 4. 系统业务流程
系统核心的“分析-检索-生成”流水线如下：
1. **输入端**: 用户通过 React 前端采集病变区域图像并上传。
2. **感知层**: FastAPI 调用 DermFM-Zero 处理图像，返回疾病分类标签及概率。
3. **检索层**: 根据疾病标签，在 FAISS 向量数据库中检索高度相关的专业医学百科切片。
4. **决策层**: LLM (Qwen) 接收“用户背景 + 视觉标签 + 医学百科”作为输入上下文。
5. **输出端**: 生成符合 HTML 规范的结构化 JSON/Markdown 报告，实时推向前端展示。

## 5. 设计哲学
- **移动优先 (Mobile First)**: 针对手机摄像头优化，界面灵动且易于操作。
- **专业驱动 (Medical Integrity)**: 拒绝盲目生成的 AI 建议，坚持以知识库检索为准绳。
- **隐私保护**: 采用 JWT 加密通信，支持本地数据持久化及 PWA 安全沙盒存储。
