# 肤理通（DermaScan AI）—— 项目工作内容总结

## 一、题目简介

**肤理通**是一个基于人工智能的**皮肤病变智能分析平台**，结合计算机视觉模型（DermFM-Zero）与大语言模型（通义千问 Qwen），为用户提供初步皮肤健康评估与建议。

技术架构为 **前后端分离**：
- **前端**: React + TypeScript + Vite + TailwindCSS（移动优先设计 / PWA）
- **后端**: Python FastAPI + SQLite + 阿里云百炼 DashScope（大模型 API）+ RAG 知识库

---

## 二、本项目完成的工作（不少于 3 点）

### 1. 皮肤病变识别与 AI 诊断分析系统

> **核心功能**：构建了完整的"拍照/上传 → AI 识别 → 报告生成"流程。

- **图像采集**：支持手机相机实时拍摄（`react-webcam`）及本地图片上传，实现一键皮肤图像获取。
- **AI 识别引擎**：后端集成 DermFM-Zero 计算机视觉模型（`dermfm_service.py`），对上传的皮肤图像进行病变分类与风险评估。
- **智能报告生成**：通过阿里云通义千问大模型（`qwen_service.py`），结合 AI 识别结果和用户个人信息（年龄、性别等），生成结构化的 HTML 格式健康分析报告，包含诊断结果、风险等级、推荐治疗方案等内容。
- **报告导出**：支持将分析报告直接导出为 PDF 文件供用户保存。

### 2. RAG 增强的皮肤医学知识库与 AI 护肤助手

> **知识服务**：构建了基于 RAG（检索增强生成）技术的医学知识服务体系。

- **RAG 知识库**：后端搭建了 FAISS 向量索引的皮肤病知识库（`rag_service.py`），包含专业皮肤病百科文章，用于增强 AI 回答的专业性和准确性。
- **皮肤百科模块**：前端提供了可浏览的皮肤病百科文章列表（`KnowledgeBase.tsx` + `ArticleDetail.tsx`），用户可按分类查询常见皮肤病的症状、成因、治疗建议等。
- **AI 护肤助手**：集成了基于大模型的在线聊天助手（`MedicalAssistant.tsx` + `chat.py`），用户可随时就皮肤问题进行交互式咨询，AI 回答结合 RAG 知识库内容，提供更专业的建议。

### 3. 用户管理与健康档案系统

> **数据管理**：完整的用户体系和健康数据持久化。

- **用户认证**：实现了完整的注册 / 登录 / JWT Token 认证体系（`AuthContext.tsx` + `user_service.py`），保障用户数据隐私。
- **个人健康档案**：支持用户维护个人信息（年龄、性别、过敏史等），并自动将这些信息融入 AI 诊断报告中，实现个性化分析。
- **分析历史记录**：所有分析结果自动保存至数据库，用户可通过历史记录页面（`History.tsx`）随时回顾过往诊断报告。
- **健康日历**：提供健康日历视图（`HealthCalendar.tsx`），以时间线方式可视化展示用户的皮肤健康变化趋势。

### 4. 移动端适配与 PWA 渐进式 Web 应用

> **用户体验**：针对移动端深度优化，支持安装为原生应用体验。

- **响应式设计**：所有页面针对手机端进行了深度适配，包含安全区域处理（刘海屏、底部手势栏）、滚动容器优化等。
- **PWA 支持**：通过 `vite-plugin-pwa` 实现离线缓存和桌面安装能力，用户可将应用添加到手机主屏幕，获得接近原生 App 的体验。
- **精品 UI 设计**：采用现代化的 UI 设计体系（绿色主题/渐变配色/动效动画），经过多轮 UI 改版迭代，打造高级感的视觉体验。
- **局域网访问**：支持通过局域网 IP 在多设备间访问，方便移动端调试和实际使用。

### 5. 后台管理与系统运维

> **系统支撑**：管理端和工程化能力保障系统稳定运行。

- **管理后台**：实现了管理员页面（`admin/`），支持用户管理和系统数据查看。
- **本地存储方案**：移除了外部对象存储（MinIO）依赖，转为本地文件系统存储（`storage_service.py`），降低了部署复杂度。
- **AI 服务迁移**：完成了从 OpenRouter 到阿里云百炼（DashScope）的 AI 服务迁移，解决了模型配额和兼容性问题。
- **数据库设计**：使用 SQLite 进行数据持久化，设计了用户表、分析记录表、知识库表等数据模型，并完成了多次数据库迁移。

---

## 三、总结

本项目围绕"皮肤健康智能分析"这一核心场景，完成了 **AI 智能诊断**、**RAG 知识库与问答助手**、**用户健康档案管理**、**移动端 PWA 体验优化**、**后台管理与系统架构** 五个大方向的工作，构建了一个功能完整、体验优良的 AI 皮肤健康分析平台。
